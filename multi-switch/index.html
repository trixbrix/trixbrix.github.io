All-In-One Controller 2.0…

Battery-powered Bluetooth servo controller for LEGO train switches. Plug it into your computer via USB and click Connect to install the latest firmware.

Unsupported browser

Firmware flashing requires the WebSerial API, which is only available in Chromium-based browsers (Chrome, Edge, Brave, Arc, Opera). Please open this page in one of those.

Install firmware

Connect the device to your computer with a USB cable, then click the button below. You'll be asked to pick the serial port for the device.

Advanced: factory reset (full erase)
⚠️ This will wipe all saved settings on the device: working mode, paired LEGO Powered Up network ID, switch color, port assignments. The device will reboot as if brand new. Only use this for a factory reset or to recover from a corrupted state.
