All-In-One Controller 2.0…

Battery-powered Bluetooth servo controller for LEGO train switches. Follow the three steps below to install the latest firmware on your controller.

This browser can't update devices

The update tool only works in Google Chrome or Microsoft Edge. Please open this page in one of those, then come back here.

Install firmware

  1. Plug your controller into the computer with the USB cable.
  2. Click Install firmware.
  3. A small window will pop up asking you to pick a device — choose the one that appeared when you plugged in (usually the only option), then click Connect.

Tip: if you see more than one option in the list, look for a name containing USB-SERIAL, CP210x, or CH340.

⚠️
This wipes all saved settings. Working mode, paired LEGO Powered Up network ID, switch color, and port assignments will be erased. The device boots up as if brand new. Only use this for a true factory reset or to recover from a corrupted state.
