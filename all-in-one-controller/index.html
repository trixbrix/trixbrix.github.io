All-In-One Controller 2.0…

Battery-powered Bluetooth servo controller for LEGO train switches. Plug it into your computer with a USB cable, then click Install firmware below.

Heads up — this browser can't flash devices

USB flashing needs the WebSerial API, which only works in Chromium-based browsers (Chrome, Edge, Brave, Arc, Opera). Reopen this page in one of those to continue.

Install firmware

Connect the device to your computer with a USB cable, then click the button. You'll be asked to pick the serial port — look for one labeled USB-SERIAL, CP210x, or CH340.

⚠️
This wipes all saved settings. Working mode, paired LEGO Powered Up network ID, switch color, and port assignments will be erased. The device boots up as if brand new. Only use this for a true factory reset or to recover from a corrupted state.
